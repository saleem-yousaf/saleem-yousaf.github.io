// the two failures I see most

Where the edge quietly stops working

  • Origin not locked down. The WAF is decorative if the origin still answers the internet.
  • WAF still in detection mode. Detection logs an attack. Prevention stops one.
  • Caching a response that ignores the auth header.
  • Health probes that return healthy while the app is broken.
  • Confusing global with regional. Front Door is not App Gateway.
  • Forgetting the ACM cert and WAF ACL must live in us-east-1.
